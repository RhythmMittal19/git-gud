## Foundation Track

> **Timeline:** Weeks 1-4 intensive, then ongoing daily practice

> **Why This Matters:** These skills multiply ALL other learning. Skip this and you'll struggle forever.

This track runs BEFORE and PARALLEL to everything else. These skills multiply all other learning.

**The Math:**

- Typing at 40 WPM vs 80 WPM = 2x speed difference

- Linux knowledge = Understand every server you'll ever touch
