### F1: Touch Typing (Weeks 1-8)

> **Why this matters:** You'll type millions of lines in your career. 40 WPM vs 80 WPM is a 2x productivity difference. Wrong muscle memory now means fighting your own hands forever.

> **Time Investment:** ~30 min/day | **Total:** ~28 hours over 8 weeks

My typing is currently at 15-45 WPM with wrong hand placement. This is costing thousands of hours over a career.

**The Problem:**

- Wrong muscle memory is harder to fix than starting fresh
- You must UNLEARN before you learn
- This will feel slow and frustrating for 2-3 weeks
- Your WPM will DROP before it rises — this is normal and expected

**The Method:**

**Week 1-2: Complete Reset**

```text
1. Learn proper hand position (home row: A S D F — J K L ;)
2. Left pinky: A, Right pinky: ;
3. Left ring: S, Right ring: L
4. Left middle: D, Right middle: K
5. Left index: F (and G), Right index: J (and H)
6. Thumbs: Spacebar

Rules:
- Practice ONLY with proper technique, even if slower
- Do NOT look at keyboard — ever (cover it if needed)
- Accept that you'll drop to 15-20 WPM temporarily
- Focus on accuracy over speed (95%+ accuracy first)
```

**Week 3-4: Building Speed**

```text
1. Continue proper technique — no cheating
2. Aim for 98%+ accuracy before pushing speed
3. Speed comes naturally once accuracy is solid
4. Practice 30 minutes minimum daily (consistency > duration)
5. Track your progress daily
```

**Week 5-8: Programmer-Specific**

```text
1. Practice symbols: { } [ ] ( ) < > ! @ # $ % ^ & * _ - + =
2. Practice code snippets (not just English text)
3. Practice common patterns: function, const, return, if/else
4. Target: 60 WPM with 98% accuracy by Week 8
```

**Daily Practice Schedule:**

```text
Morning (optional):  5-10 min warmup on Keybr
Evening:            20-25 min focused practice on TypingClub
                    5 min speed test on MonkeyType
```
