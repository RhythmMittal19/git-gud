# Using Accessibility Features

## ♿ Tools for Disabilities

Accessibility features help people with visual, hearing, or mobility limitations.

### Common Features

- **Blind/Low-Vision:**
  - **Text-to-Speech:** Reads screen text aloud.
  - **High Contrast:** Changes colors to make text easier to see.
  - **Screen Magnifiers:** Zooms in on the screen.
- **Hearing:**
  - **Closed-Captioning:** Visual text for audio.
  - **Mono Audio:** Sends right/left signals to both ears.
- **Mobility:**
  - **Sticky Keys:** Press shortcuts (like `Ctrl+Alt+Del`) one key at a time.
  - **Speech Recognition:** Control the computer with voice.

### How to Access

- **Windows:** Settings > Ease of Access.
- **Mac:** System Preferences > Accessibility.
- **Web Browsers:** Use `Ctrl +` (PC) or `Cmd +` (Mac) to zoom in on web pages.
