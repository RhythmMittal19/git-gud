# Getting to Know the OS

## 🗂️ File Systems

Computers use folders to organize files.

- **Windows:** Uses **File Explorer** (Folder icon).
- **Mac:** Uses **Finder** (Face icon on the Dock).

### Common Actions

- **Opening Files:** Double-click a file. The computer automatically chooses the right app to open it.
- **Deleting Files:** Files go to the **Recycle Bin** (PC) or **Trash** (Mac). To permanently delete, you must empty the bin.
- **Settings:**
  - **Windows:** Start > Settings (or Control Panel).
  - **Mac:** Apple Icon > System Preferences.

### Shutting Down

Always shut down properly to protect your data.

- **Windows:** Start > Power > Shut Down.
- **Mac:** Apple Icon > Shut Down.
