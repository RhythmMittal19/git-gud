# Getting Started with Your First Computer

## 🚀 The Basics of Usage

### 1. Turning It On

Press the power button (universal circle/line symbol). The computer will **boot up** (takes 15 seconds to a few minutes). You may need to log in with a user profile and password.

### 2. Using the Mouse/Trackpad

- **Click:** Select an object.
- **Double-click:** Open a file or program.
- **Drag:** Move items around.
- **Cursor:** The arrow on the screen.

### 3. The Desktop

The main screen is called the desktop.

- **Icons:** Small images representing files or commands.
- **Menus:** Organized collections of commands.
- **Windows:** Contained areas for open programs. You can switch between windows to multitask.
