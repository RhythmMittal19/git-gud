# Connecting to the Internet

## 🌐 Internet Service Types

You typically purchase internet from an ISP (Internet Service Provider).

1.  **Dial-up:** Slowest. Uses phone lines. Cannot use phone and internet at the same time.
2.  **DSL:** Broadband via phone line. Faster than dial-up; allows phone usage simultaneously.
3.  **Cable:** Uses cable TV lines. Broadband. Fast.
4.  **Satellite:** Connects via satellites. Good for rural areas, but weather can affect it.
5.  **3G/4G/5G:** Cellular networks for mobile phones.

## Hardware Needed

- **Modem:** The primary device. Type depends on service (Cable modem, DSL modem).
- **Router:** Allows you to create a **Home Network (Wi-Fi)** so multiple devices can connect wirelessly.
