# Getting Started with the Internet

## 🌍 Navigating the Web

The Internet is the global network; the Web is the collection of websites.

### Key Concepts

- **Web Browser:** The application used to view sites (Chrome, Safari, Firefox).
- **URL:** The unique address of a website (e.g., `www.google.com`).
- **Hyperlinks:** Text (usually blue/underlined) that links to other pages.

### Search Engines

Specialized websites (Google, Bing, Yahoo) designed to help you find information. Type **keywords** to find relevant results.

### What Can You Do?

- **Email:** Communicate instantly.
- **Social Media:** Connect with communities.
- **Streaming:** Watch TV/Movies.
- **Education:** Learn new skills online.
