# Understanding Operating Systems (OS)

## 👮 The Manager

The Operating System (OS) manages all hardware and software. It coordinates the CPU, memory, and storage to make sure every program gets what it needs.

> **[Video: Watch the video below to learn more about operating systems.]**

### Types of OS

Most computers use a **GUI (Graphical User Interface)**, allowing you to click icons and buttons.

1.  **Microsoft Windows:** Created in the mid-1980s. The most popular OS in the world. Versions include Windows 10, 8, 7, and Vista.
2.  **macOS:** Created by Apple. Known for its look and feel. Accounts for less than 10% of global users.
3.  **Linux:** An open-source, free operating system. Often used on servers. Variations include Ubuntu, Linux Mint, and Fedora.

### Mobile OS

Designed specifically for smaller screens:

- **Apple iOS:** Used on iPhone/iPad.
- **Google Android:** Used on many different phone brands.
- **Limitations:** Mobile OSs cannot run full desktop software, but use "Apps" instead.
