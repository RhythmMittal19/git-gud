# Understanding Applications

## 🛠️ What is an App?
An **App** (Application) is software that performs specific tasks.
*   **Multitasking:** Running more than one app at the same time.

### Desktop Applications
*   **Word Processors:** (Microsoft Word) Write letters, design flyers.
*   **Web Browsers:** (Chrome, Firefox, Safari) Access the internet.
*   **Media Players:** (iTunes, Windows Media Player) Listen to music or watch movies.
*   **Games:** (Solitaire, Halo) Entertainment.

### Mobile Apps
*   **Gmail:** View and send email.
*   **Instagram:** Share photos.
*   **Duolingo:** Learn languages.

You can install new applications from the web or app stores to add functionality to your device.
