# Inside a Computer

## 🧠 The Internal Components

If you look inside a computer case, you will see several key components connected to the motherboard.

> **[Video: Watch the video below to learn about what's inside a desktop computer.]**

### 1. Motherboard

The main circuit board. It is a thin plate that holds the CPU, memory, and connectors. It connects everything together.

### 2. CPU (Central Processing Unit)

Also called the **Processor**. It is the "brain" of the computer inside a 2-inch ceramic square.

- **Speed:** Measured in **MHz** (millions of instructions/sec) or **GHz** (billions of instructions/sec).
- **Heat Sink:** Sits on top of the CPU to absorb heat.

### 3. RAM (Random Access Memory)

**Short-term memory.** Whenever your computer performs calculations, it stores data here temporarily.

- **Volatile:** RAM clears when you turn the computer off. You must save work to the hard drive to keep it.
- **Performance:** More RAM allows you to run more programs at once without sluggishness.

### 4. Hard Drive

**Long-term storage.** Holds software, documents, and files. Data is safe even when power is off.

### 5. Expansion Cards

Cards added to the motherboard to boost performance:

- **Video Card (GPU):** Responsible for what you see on the screen.
- **Sound Card:** Responsible for audio to speakers.
- **Network Card:** Allows Wi-Fi or Ethernet access.
- **Bluetooth Card:** For wireless mice/keyboards.
