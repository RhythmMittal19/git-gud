The Box Model

Understanding how width and height are calculated.

1. Content Box (Default)

box-sizing: content-box;

Width (200) + Padding (40) + Border (20) = 260px Total

I grow bigger than 200px!

2. Border Box (Recommended)

box-sizing: border-box;

Total Width is forced to be 200px.

I stay exactly 200px.
