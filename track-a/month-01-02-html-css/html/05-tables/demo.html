📊 05 - Tables

Tables organize data into rows and columns. Use them for tabular data only (not for layout!).

📋 Basic Table Structure

┌─────────────────────────────────────────────────────────────┐ │ <table> │ │ │ │ │ ├── <thead> ← Table Header (column titles) │ │ │ └── <tr> ← Table Row │ │ │ ├── <th>Header 1</th> ← Table Header Cell │ │ │ ├── <th>Header 2</th> │ │ │ └── <th>Header 3</th> │ │ │ │ │ └── <tbody> ← Table Body (actual data) │ │ │ ├── <tr> ← Row 1 │ │ │ │ ├── <td>Data 1</td> ← Table Data Cell │ │ │ │ ├── <td>Data 2</td> │ │ │ │ └── <td>Data 3</td> │ │ └── <tr> ← Row 2 │ │ │ ├── <td>Data 4</td> │ │ │ ├── <td>Data 5</td> │ │ │ └── <td>Data 6</td> │ └─────────────────────────────────────────────────────────────┘

Simple Table Example

CODE
<table>
<thead>
<tr>
<th>Name</th>
<th>Age</th>
<th>City</th>
</tr>
</thead>
<tbody>
<tr>
<td>Rhythm</td>
<td>21</td>
<td>Delhi</td>
</tr>
<tr>
<td>John</td>
<td>25</td>
<td>Mumbai</td>
</tr>
</tbody>
</table>
OUTPUT
Name Age City
Rhythm 21 Delhi
John 25 Mumbai

📦 Table Elements Reference

Element Name Purpose
<table> Table Container for the entire table
<thead> Table Head Groups header content
<tbody> Table Body Groups body content (main data)
<tfoot> Table Foot Groups footer content (totals, etc.)
<tr> Table Row A horizontal row
<th> Table Header Cell Header cell (bold, centered by default)
<td> Table Data Cell Regular data cell
<caption> Caption Title for the table

📊 Table with Caption and Footer

CODE
<table>
<caption>Monthly Sales Report</caption>
<thead>
<tr>
<th>Month</th>
<th>Sales</th>
</tr>
</thead>
<tbody>
<tr>
<td>January</td>
<td>₹50,000</td>
</tr>
<tr>
<td>February</td>
<td>₹65,000</td>
</tr>
</tbody>
<tfoot>
<tr>
<th>Total</th>
<td>₹1,15,000</td>
</tr>
</tfoot>
</table>
OUTPUT
Monthly Sales Report
Month Sales
January ₹50,000
February ₹65,000
Total ₹1,15,000

🔀 Spanning Rows and Columns

colspan - Merge Columns

CODE
<table>
<tr>
<th colspan="3">Student Details</th>
</tr>
<tr>
<th>Name</th>
<th>Age</th>
<th>Grade</th>
</tr>
<tr>
<td>Rhythm</td>
<td>21</td>
<td>A</td>
</tr>
</table>
OUTPUT
Student Details
Name Age Grade
Rhythm 21 A

rowspan - Merge Rows

CODE
<table>
<tr>
<th>Category</th>
<th>Item</th>
</tr>
<tr>
<td rowspan="2">Fruits</td>
<td>Apple</td>
</tr>
<tr>
<td>Banana</td>
</tr>
<tr>
<td rowspan="2">Vegetables</td>
<td>Carrot</td>
</tr>
<tr>
<td>Potato</td>
</tr>
</table>
OUTPUT
Category Item
Fruits Apple
Banana
Vegetables Carrot
Potato
💡 Remember:
  • colspan="2" = Cell spans 2 columns horizontally →
  • rowspan="2" = Cell spans 2 rows vertically ↓

⚠️ Common Mistakes

❌ Using tables for layout

Tables are for DATA only, not for page layout. Use CSS Flexbox/Grid for layouts.

❌ Missing thead/tbody

Always use <thead> and <tbody> for proper structure and accessibility.

❌ Using <td> for headers

Use <th> for header cells, not <td>. Screen readers need this distinction.
