🔗 03 - Links & Images

Links connect pages together. Images make content visual. Both are essential for any website.

🔗 Links (Anchor Tags)

The <a> tag creates hyperlinks.

Basic External Link

CODE
<a href="https://github.com">Go to GitHub</a>
OUTPUT
Go to GitHub

Open in New Tab

CODE
<a href="https://github.com" target="_blank"> GitHub (new tab) </a>
OUTPUT
GitHub (new tab)
💡 target="_blank" opens link in a new browser tab. Use for external websites so users don't leave your site.

All Types of Links

Type Code Use Case
External Link <a href="https://google.com"> Link to other websites
Internal Link <a href="about.html"> Link to pages on your site
Anchor Link <a href="#section-id"> Jump to section on same page
Email Link <a href="mailto:me@email.com"> Opens email client
Phone Link <a href="tel:+1234567890"> Opens phone dialer (mobile)
Download Link <a href="file.pdf" download> Downloads the file

Anchor Links (Jump to Section)

How Anchor Links Work
Step 1: Create a link with # and an ID ┌─────────────────────────────────────────┐ │ <a href="#contact">Go to Contact</a> │ └─────────────────────────────────────────┘ │ Clicking this jumps to... ▼ Step 2: An element with matching ID ┌─────────────────────────────────────────┐ │ <section id="contact"> │ │ <h2>Contact Us</h2> │ │ </section> │ └─────────────────────────────────────────┘

🖼️ Images

The <img> tag displays images. It's a self-closing tag (no </img> needed).

Basic Image

CODE
<img src="https://via.placeholder.com/300x150" alt="Placeholder image" >
OUTPUT

Important Image Attributes

Attribute Purpose Required?
src Image source (URL or file path) ✅ Yes
alt Alternative text (accessibility, SEO) ✅ Yes
width Width in pixels Optional
height Height in pixels Optional
loading="lazy" Load image only when visible Optional (recommended)
⚠️ Always include alt text!
  • Screen readers read it aloud (accessibility)
  • Shows if image fails to load
  • Helps with SEO
  • Be descriptive: "Golden retriever playing fetch" NOT "dog.jpg"

Setting Image Size

CODE
<img src="https://via.placeholder.com/400x200" alt="Sized image" width="200" height="100" >
OUTPUT (200x100)

Image as a Link

CODE
<a href="https://github.com" target="_blank"> <img src="https://via.placeholder.com/150x80" alt="Click me!" > </a>
OUTPUT (clickable!)

👆 Click the image!

Figure with Caption

CODE
<figure> <img src="https://via.placeholder.com/300x150" alt="A beautiful sunset" > <figcaption> A beautiful sunset over the ocean </figcaption> </figure>
OUTPUT
💡 Use <figure> and <figcaption> for images that need captions. It's semantic HTML and better for accessibility.

📂 Understanding File Paths

Your Project Folder: ┌─────────────────────────────────────────┐ │my-website/ | │ ├── index.html ◄── You are here | │ ├── about.html | │ └── pages/ | │ │ └──contact.html | │ ├──images/ | │ │ ├── logo.png | │ │ └── hero.jpg | └─────────────────────────────────────────┘ From index.html, how to link to: ───────────────────────────────────────── logo.png → "images/logo.png" about.html → "about.html" contact.html → "pages/contact.html" From contact.html, how to link to: ───────────────────────────────────────── index.html → "../index.html" (..means go up one folder) logo.png → "../images/logo.png"
Path Type Example Meaning
Same folder about.html File in same folder
Subfolder images/logo.png Inside images folder
Parent folder ../index.html Go up one level
Absolute URL https://site.com/image.jpg Full web address
Root relative /images/logo.png From website root
