📋 04 - Lists

HTML provides three types of lists for organizing content.

● Unordered List (Bullets)

Use <ul> when order doesn't matter (like a shopping list).

CODE
<ul>
<li>Apples</li>
<li>Bananas</li>
<li>Oranges</li>
</ul>
OUTPUT
  • Apples
  • Bananas
  • Oranges

1. Ordered List (Numbers)

Use <ol> when order DOES matter (like recipe steps).

CODE
<ol>
<li>Preheat oven to 350°F</li>
<li>Mix ingredients</li>
<li>Bake for 30 minutes</li>
</ol>
OUTPUT
  1. Preheat oven to 350°F
  2. Mix ingredients
  3. Bake for 30 minutes

Ordered List Attributes

Attribute Code Result
Start from 5 <ol start="5"> 5, 6, 7...
Reverse <ol reversed> 3, 2, 1...
Letters <ol type="A"> A, B, C...
Lowercase <ol type="a"> a, b, c...
Roman <ol type="I"> I, II, III...
CODE
<ol type="A">
<li>First item</li>
<li>Second item</li>
<li>Third item</li>
</ol>
OUTPUT
  1. First item
  2. Second item
  3. Third item

📦 Nested Lists

Put lists inside lists for hierarchy.

CODE
<ul>
<li>Frontend
<ul>
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>
</ul>
</li>
<li>Backend
<ul>
<li>Node.js</li>
<li>Python</li>
</ul>
</li>
</ul>
OUTPUT
  • Frontend
    • HTML
    • CSS
    • JavaScript
  • Backend
    • Node.js
    • Python

📖 Description List

Use <dl> for term-definition pairs (like a glossary).

CODE
<dl>
<dt>HTML</dt>
<dd>HyperText Markup Language</dd>
<dt>CSS</dt>
<dd>Cascading Style Sheets</dd>
<dt>JS</dt>
<dd>JavaScript</dd>
</dl>
OUTPUT
HTML
HyperText Markup Language
CSS
Cascading Style Sheets
JS
JavaScript
💡 Description List Elements:
  • <dl> = Description List (container)
  • <dt> = Description Term (the word/term)
  • <dd> = Description Details (the definition)

🤔 When to Use Which?

Quick Reference
List Type Use When Examples
<ul> Order doesn't matter Shopping list, features, menu items
<ol> Order DOES matter Steps, rankings, instructions
<dl> Term + Definition pairs Glossary, FAQs, metadata

⚠️ Common Mistakes

❌ Wrong: Content directly in ul/ol

<ul> Item 1 <!-- WRONG: No <li> tag --> Item 2 </ul>

✅ Correct: Always use <li>

<ul> <li>Item 1</li> <li>Item 2</li> </ul>
