📄 01 - HTML Document Structure

Every HTML page needs a proper structure. This is the foundation of all web pages.

🏗️ The Basic Template

Every HTML file should have this structure:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Page Title</title> </head> <body> <!-- Your content goes here --> </body> </html>

🔍 What Each Part Does

Element Purpose Required?
<!DOCTYPE html> Tells browser "this is HTML5" ✅ Yes
<html lang="en"> Root element, sets language ✅ Yes
<head> Contains metadata (not visible) ✅ Yes
<meta charset="UTF-8"> Allows all characters & emojis 🎉 ✅ Yes
<meta name="viewport"...> Makes page work on mobile ✅ Yes
<title> Text shown in browser tab ✅ Yes
<body> All visible content goes here ✅ Yes

⚠️ Common Mistakes

❌ Mistake 1: Missing DOCTYPE

<html> <head>...</head> <body>...</body> </html> // Missing <!DOCTYPE html> at the top!
Why it matters: Browser might render page in "quirks mode" which can cause layout issues.

❌ Mistake 2: Content in Head

<head> <title>My Page</title> <p>Hello World</p> ← WRONG! This is visible content </head>
Why it matters: Visible content belongs in <body>, not <head>.

❌ Mistake 3: Missing Language

<html> ← Missing lang="en"
Why it matters: Screen readers won't know how to pronounce words. Bad for accessibility.

✅ Correct Template (Copy This!)

This is what every HTML file should look like:
<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Your Page Title</title> </head> <body> <h1>Your Main Heading</h1> <p>Your content goes here.</p> </body> </html>
