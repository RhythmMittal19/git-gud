### Structured 6-Month Path

> **For those who want an exact curriculum:**

```text
MONTH 1: Foundations
Week 1-2: HTML (The Odin Project: Foundations)
Week 3-4: CSS (The Odin Project + Kevin Powell YouTube)
Daily: Typing practice (30 min)

MONTH 2: JavaScript
Week 1-2: JS Basics (JavaScript.info Part 1)
Week 3-4: DOM & Events (JavaScript.info Part 2)
Projects: Calculator, Todo List
Daily: DSA Easy (30 min, 5 problems/week)

MONTH 3: Advanced JS & TypeScript
Week 1-2: Async JS (JavaScript.info Part 1 continued)
Week 3-4: TypeScript (Official Handbook)
Projects: Weather App, GitHub Finder
Daily: DSA Easy/Medium (1 hour)

MONTH 4: React
Week 1-2: React Basics (react.dev)
Week 3-4: React + TypeScript + Router
Projects: Full React app with routing
Daily: DSA Medium (1 hour)
Start: Job applications (10/day)

MONTH 5: Backend & Full Stack
Week 1-2: Node.js + Express
Week 3-4: PostgreSQL + Prisma
Projects: REST API, Full-stack app
Daily: DSA Medium (1.5 hours)
Continue: Job applications (15/day)

MONTH 6: Next.js & Job Hunt
Week 1-2: Next.js (official learn course)
Week 3-4: Portfolio projects + polish
Focus: Interviews, mock interviews
Daily: DSA Medium/Hard (1.5 hours)
Continue: Job applications (20/day)
```

### Building a Learning System

**Don't just consume content. Build a system:**

```text
1. SCHEDULE
   - Same time every day
   - Protect this time
   - Start small (1 hour) and grow

2. TAKE NOTES
   - Obsidian or Notion
   - Write in your own words
   - Include code examples
   - Link related concepts

3. BUILD PROJECTS
   - After each major topic
   - Don't just copy tutorials
   - Modify and extend
   - Deploy and share

4. TEACH WHAT YOU LEARN
   - Twitter threads
   - Blog posts
   - Explain to friends
   - Rubber duck debugging

5. REVIEW REGULARLY
   - Weekly: What did I learn?
   - Monthly: What are my gaps?
   - Quarterly: Am I on track?

6. ADJUST AS NEEDED
   - Not working? Try different resource
   - Going too slow? Focus and remove distractions
   - Going too fast? Slow down and build more
```
